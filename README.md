## 💻 Développement Local

1. Cloner le repository :
```bash
git clone https://github.com/Kamalghalloussidev/coffree-fr.git
cd coffree-fr
```

2. Ouvrir `index.html` dans un navigateur ou démarrer un serveur local :
```bash
python -m http.server 8000
```

3. Ouvrir [http://localhost:8000](http://localhost:8000)
