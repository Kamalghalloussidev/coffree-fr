# COFFREE.FR

Site et application mobile en cours de développement.

## 📋 Description

COFFREE.FR est une plateforme innovante qui sera bientôt disponible sur le web ainsi que sur les plateformes mobiles Android et iOS.

## 🎨 Design

- **Couleur principale**: Vert (#2d5016 - #4a7c2c)
- **Couleur secondaire**: Beige (#f5f5dc)
- **Style**: Moderne avec effet glassmorphism

## 📱 Plateformes

- ✅ Site Web
- 🔜 Application Android
- 🔜 Application iOS

## 🏗️ Architecture du Projet

```
Projet COFFREE.FR/
├── index.html              # Page d'accueil principale
├── assets/                 # Ressources du site
│   ├── css/               # Feuilles de style
│   │   └── style.css      # Style principal
│   ├── images/            # Images et photos
│   │   └── logo.jpeg      # Logo COFFREE.FR
│   ├── svg/               # Icônes et graphiques SVG
│   │   ├── android-logo-2.svg
│   │   └── apple-13.svg
│   └── js/                # Scripts JavaScript
│       └── main.js        # Script principal
└── docs/                  # Documentation
    └── README.md          # Ce fichier
```

## 🚀 Installation

1. Cloner ou télécharger le projet
2. Ouvrir `index.html` dans un navigateur web
3. C'est tout ! Aucune dépendance requise

## 💻 Technologies Utilisées

- HTML5
- CSS3 (avec animations et glassmorphism)
- JavaScript Vanilla
- SVG pour les icônes
