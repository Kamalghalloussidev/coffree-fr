# 🏗️ Architecture du Projet COFFREE.FR

## 📁 Structure Complète

```
Projet COFFREE.FR/
│
├── 📄 index.html                    # Page d'accueil (en construction)
├── 📄 .gitignore                    # Configuration Git
│
├── 📂 assets/                       # Ressources du site
│   │
│   ├── 📂 css/                      # Styles CSS
│   │   └── 📄 style.css            # Feuille de style principale
│   │
│   ├── 📂 images/                   # Images et photos
│   │   └── 🖼️ logo.jpeg            # Logo COFFREE.FR
│   │
│   ├── 📂 svg/                      # Icônes vectorielles
│   │   ├── 📄 android-logo-2.svg   # Logo Android
│   │   └── 📄 apple-13.svg         # Logo Apple iOS
│   │
│   └── 📂 js/                       # Scripts JavaScript
│       └── 📄 main.js              # Script principal avec animations
│
└── 📂 docs/                         # Documentation
    └── 📄 README.md                # Documentation complète du projet
```

## 📋 Description des Dossiers

### `/assets/`
Contient toutes les ressources statiques du site web.

#### `/assets/css/`
- Feuilles de style CSS
- Design vert et beige
- Animations et effets visuels
- Responsive design

#### `/assets/images/`
- Logo et images du site
- Format JPEG/PNG pour les photos
- Optimisées pour le web

#### `/assets/svg/`
- Icônes vectorielles (SVG)
- Logos des plateformes mobiles
- Graphiques scalables sans perte de qualité

#### `/assets/js/`
- Scripts JavaScript
- Animations et interactivité
- Gestion des événements

### `/docs/`
Documentation technique et informations sur le projet.

## 🎯 Utilisation

1. **Page d'accueil**: `index.html` (ouvrir dans un navigateur)
2. **Styles**: Modifier `assets/css/style.css`
3. **Scripts**: Modifier `assets/js/main.js`
4. **Documentation**: Consulter `docs/README.md`

## 🔧 Technologies

- **HTML5**: Structure sémantique
- **CSS3**: Design moderne avec glassmorphism
- **JavaScript**: Interactivité et animations
- **SVG**: Icônes vectorielles

## 📱 Responsive

Le site s'adapte automatiquement à tous les écrans:
- 📱 Mobile (320px+)
- 📱 Tablette (768px+)
- 💻 Desktop (1024px+)
